# Terminal Ark Changelog

## APP BUNDLE VERSION v0.6.3 Public Beta
## SERVER BUNDLE VERSION v0.6.3 Public Beta

## Release Date: Apr. 5th 2026 20:30 JST

### Changelog
- Added new execute plan in execute machine called massive production, you can destory items here with high products droprates.
- Execute plan in execute machine now can be skipped via Overclocking.
- Execute machine is only available to players in Terminal User Network.
- AFK exp now calculated with the overall level, if your overall level is higher then area max level, area max level will be choosen to calculate exp.
- User docs will be synced using realtime, no need to refresh manual or by loop request, so manual refresh by clicking HomeUserBar is removed.
- Added new way to use neural regen items to HomeUserBar by adding a plus icon in the end.
- Added physical/magic affinity display for heroes and dynamic HP bar coloring based on affinity, bringing hero cards in line with enemy display style.
- Fixed contact center slot incorrectly showing "Offline" during an active agent conference; now displays "Conference Active" instead.
- Conference clue token generation now scales dynamically with agent mood (+2–5/hr with agent, +1/hr without), and agents now lose mood during conferences.
- Personal buffs are now clickable to view details, matching the global buff interaction pattern, with translations added for all four languages.

### Others
- Included various minor fixes and performance optimizations.

***This is a hotfix update and will be installed automatically. You will receive a notification prompting you to refresh the page after the hotfix is installed.***
