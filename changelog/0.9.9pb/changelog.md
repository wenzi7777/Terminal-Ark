# Terminal Ark Changelog

## APP BUNDLE VERSION v0.9.9 Public Beta
## SERVER BUNDLE VERSION v0.9.9 Public Beta

## Release Date: Jul. 17th 2026 22:40 JST

### Changelog
- Fixed synchronization issues in multiplayer dungeons.
Added a notification when a profile does not have enough required items.
- Fixed several UI issues in the Perform AFK component.
- Fixed several highlighting issues in the beginner tutorial.
- Fixed various issues in the dungeon and multiplayer dungeon systems.
- Reworked the Flee mechanic. Fleeing no longer kills a hero, but now has a small chance of succeeding. If the attempt fails, the turn is skipped.
- Fixed an issue where players could leave a dungeon while still in battle.
- Redesigned the Terminal User Network page.
- Redesigned the Inventory, Heroes, and Team pages.
- Redesigned the Disc Inventory page.
- Redesigned the Disc Heroes page and hero list workflow.
- Redesigned the Disc Team page with a sticky summary and clearer section hierarchy.
- Redesigned the Disc navigation tabs with consistent segmented controls across the Heroes, Inventory, and Team pages.
- Redesigned the Hero Card component with cleaner rarity styling and improved press feedback.
- Redesigned the Hero Details view, including equipment loadout cards and sticky action tools.
- Redesigned the Skill Slot component.
- Redesigned the selection states and press feedback of the Select Equipment overlay.
- Redesigned the Team Profile Manager editor.
- Redesigned the Select Hero and Select Support Hero pickers.
- Redesigned the Auto Team and Auto Equip Preview overlays for improved visual consistency with the Disc interface.
- Improved mobile item and hero detail sheets with optional drag-to-dismiss support.
- Redesigned the text input in ChatNet.
- Redesigned the Wasteland Development Terminal.
- Redesigned the Player Market page.
- Fixed an issue where featured items were not displayed in the Signal Hijack System.
- Fixed an issue where items in the Gacha Drop Rate viewer were incorrectly categorized as 1-Star Heroes.

### Others
- Included various minor fixes and performance optimizations.
