# Terminal Ark Changelog

## APP BUNDLE VERSION v0.9.15 Public Beta
## SERVER BUNDLE VERSION v0.9.15 Public Beta

## Release Date: Jul. 22nd 2026 21:00 JST

### Changelog
- Redesigned the hero level-up interface with single-selection core cards and progressive quantity/stat previews for a simpler, cleaner experience.
- Overhauled the hero skill progression system with the new Sync Level mechanic. Class-specific skill materials can now be obtained from limited-time dungeons, replacing the previous reliance on rare Memory Fragments, which can now be exchanged for Encrypted Chip Fragments instead.
- Redesigned the Hero Level Up interface.
- Removed Challenge Dungeons from the Limited-Time Dungeons.
- Fixed a text shadow issue affecting the filter icons on the /disc/heroes page.
- Added item descriptions for all items.
- Fixed an issue where inventory upgrade items did not have the correct rarity assigned.
- Redesigned the UI of the Upgrade Market Slots page.

### Others
- Included various minor fixes and performance optimizations.

***This is a hotfix update and will be installed automatically. You will receive a notification prompting you to refresh the page after the hotfix is installed.***
