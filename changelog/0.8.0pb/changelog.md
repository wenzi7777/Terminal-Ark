# Terminal Ark Changelog

## APP BUNDLE VERSION v0.8.0 Public Beta
## SERVER BUNDLE VERSION v0.8.0 Public Beta

## Release Date: Jul. 10th 2026 07:10 JST

### Changelog
- Redesigned the dungeon system.
- Added buttons for internal links in ChatNet messages.
- Fixed an issue where equipment in the Executing Machine selection page of the Wasteland Development Terminal was not displayed one item per row.
- Improved the usability of the Text Input component.
- Fixed an issue where the icon in the General Button component had an excessively high z-index.
- Fixed an issue where the upgrade benefits of the Executing Machine in the Wasteland Development Terminal were displayed incorrectly in the upgrade preview overlay.
- Improved the time display in the Recipe Confirmation Overlay component.
- Players can now choose how many stacks of Overclocking to activate.
- ChatNet now automatically groups consecutive chat messages from the same player, up to 5 messages, as long as each message is sent within 1 minute of the previous one.
- Fixed an issue where i18n did not work correctly in some components.

### Others
- Included various minor fixes and performance optimizations.

***This is a hotfix update and will be installed automatically. You will receive a notification prompting you to refresh the page after the hotfix is installed.***
