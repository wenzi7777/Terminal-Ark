# Terminal Ark Changelog

## APP BUNDLE VERSION v0.7.3 Public Beta
## SERVER BUNDLE VERSION v0.7.3 Public Beta

## Release Date: Jul. 6th 2026 22:40 JST

### Changelog
- Fixed an issue where, in some cases, players could not deploy heroes to facilities in the Wasteland Development Terminal.
- Improved the UI of the Deploy Heroes overlay in the Wasteland Development Terminal.
- Fixed an issue where the indicator icon on the right side of the Small Chat Box would be pushed out of position when the text was too long.
- Improved item selection in the Executing Machine within the Wasteland Development Terminal.
- Fixed the Start Recycling button in the Terminal User Network. It now correctly redirects players to the Executing Machine.
- Reworked the agent skill inspection feature in the agent selection overlay. Players can now enable inspection mode and click an agent to view detailed skill information.
- Combined the Dispatch and Cancel Dispatch actions into a unified Manage Agents feature. The dedicated Cancel Dispatch button has been removed.
- Improved the UI for certain execution plans in the Executing Machine.
- Reworked the inventory expansion system. Removed Void Expansion Chips and Resonance Expansion Chips.
- Reworked the Upgrade Inventory and Upgrade Vault sections on the Data Vault page.

### Others
- Included various minor fixes and performance optimizations.

***This is a hotfix update and will be installed automatically. You will receive a notification prompting you to refresh the page after the hotfix is installed.***
