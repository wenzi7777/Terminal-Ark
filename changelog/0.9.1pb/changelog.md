# Terminal Ark Changelog

## APP BUNDLE VERSION v0.9.1 Public Beta
## SERVER BUNDLE VERSION v0.9.1 Public Beta

## Release Date: Jul. 13rd 2026 02:50 JST

### Changelog
- Fixed an issue where multiplayer dungeons did not function properly.
- Added dungeon information prefetching on the Join Dungeon page.
- Fixed an issue where the beginner tutorial did not correctly reveal the highlighted area.
- Added teammate display for multiplayer dungeons.
- Improved the UI of the Share Dungeon overlay.
- Fixed an issue where dungeon loot tiles granted too little EXP.
- Fixed an issue where dungeon enemy tiles contained only one enemy.
- Added a new Reduce Motion mode that disables unnecessary animations.
- Fixed an issue where the reward preview during dungeon exploration was displayed incorrectly.
- Added a new message reply feature to ChatNet.
- Reduced rewards from enemies in City Battles.
- Reworked the Molecular Assembler page.
- Added a series of new items for the Molecular Assembler, allowing players to specify equipment affixes when printing equipment.

### Others
- Included various minor fixes and performance optimizations.
