# Terminal Ark Changelog

## APP BUNDLE VERSION v0.3.0 Public Beta
## SERVER BUNDLE VERSION v0.3.0 Public Beta

## Release Date: Jan. 13th 2026 20:10 JST

### Bug Fixes
- Fixed a bug that Terminal User Network's invite code do not shown after clicking the Generate Invite Code button.
- Fixed an issue where dungeons could not be created while using a Support Hero.
- Fixed an issue where other level progressions could not reach the main level.
- Fixed a translation issue on the /archive page where “Status” and “MANAGE-NETWORK” were not displayed correctly.

### New Features
- Introduced new level-based scaling for EXP, Memory, and Cache drop rates and quantities.

### Improvements
- Improved the viewing experience of the Support Heroes Selector Viewer.
- Added a loading indicator to map nodes in the DungeonViewer component to prevent repeated clicks.
- Improved the level progression formula to provide a smoother yet more challenging gameplay experience.
- Improved the Discord message backup format(uid included).
- Improved the message display of the SmallChatBox, allowing it to display more information.
- Replaced fixed character truncation with responsive CSS ellipsis for item and inventory descriptions, improving layout across screen sizes and languages.

### Others
- Included various minor fixes and performance optimizations.
