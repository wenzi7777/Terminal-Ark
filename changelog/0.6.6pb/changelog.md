# Terminal Ark Changelog

## APP BUNDLE VERSION v0.6.6 Public Beta
## SERVER BUNDLE VERSION v0.6.6 Public Beta

## Release Date: Apr. 17th 2026 21:30 JST

### Changelog
- Improved the mobile UI of ModalOverlay.
- Fixed an issue where Contact Station still can obtain clues even there's no agents inside.
- Fixed the issue where Neural Sync and Agent Workstation power only updated after a refresh when countdowns ended, by making the frontend update them in realtime and aligning backend consumption with pre-settlement logic.
- Fixed missing bottom safe-area padding in multiple mobile overlays and workstation detail views under PWA mode.
- Added background dimming when TerminalAppsMenu expands to improve app list text readability over busy page backgrounds.
- Adjusted TerminalAppsMenu badge aggregation so system status alerts appear only on the system status item itself and no longer propagate to the launcher or main menu icon.
- Added level summaries to equipment InventoryItem entries.

### Others
- Included various minor fixes and performance optimizations.
