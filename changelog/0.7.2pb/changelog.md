# Terminal Ark Changelog

## SERVER BUNDLE VERSION v0.7.2 Public Beta

### Changelog
- Added multi-selection support to the Executing Machine.
- Terminal User Network benefits are now displayed on the Perform AFK page.
- Refactored the i18n system.
- Improved the onboarding tutorial for agent dispatch.
- Added output descriptions for orders in the Trading Station.
- Added drop rate displays for item destruction and recycling.
- Added a new rotating dungeon type: Storage Chip Dungeons.
- Fixed the preview issue in the Signal Retention Bay.
- Added an auto-farming system for rotating dungeons.
- Improved the input experience on the gift code redemption page.
- Fixed an issue where the current overlay would not close automatically when the Inspect Player Profile button was clicked on the Friends page.
- Removed the unused default avatar for players without an avatar on the Friends page.
- Improved the Terminal User Network page to prevent flashing while loading.
- Improved the UI design of the Player Market page to better match the game's visual style.
- Fixed an issue where new players could not change the recommended value in the Visual Effects Setup.
- Added a new Terminal Appearance for Public Beta participants.
