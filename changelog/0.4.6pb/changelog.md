# Terminal Ark Changelog

## APP BUNDLE VERSION v0.4.6 Public Beta
## SERVER BUNDLE VERSION v0.4.6 Public Beta

## Release Date: Feb. 2nd 2026 1:00 JST

### Bug Fixes
- Fixed an intermittent issue during battles that caused the error “Cannot read properties of undefined (reading 'reduce')”.

### New Features
- This update does not introduce any major new features.

### Improvements
- Improved scrollbar behavior on Windows, scrollbars now appear only when needed.
- Enhanced the GeneralButton loading animation by adding an hourglass indicator.
- The scroll position is now preserved when returning from inspecting items in your inventory.
- The TerminalAppsMenu is now displayed on all pages except /prelogin/* .
- Add a notification dot to the App Launcher icon when any launcher app has unread/check-in alerts.

### Others
- Included various minor fixes and performance optimizations.
