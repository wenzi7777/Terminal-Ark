# Terminal Ark Changelog

## APP BUNDLE VERSION v0.9.12 Public Beta
## SERVER BUNDLE VERSION v0.9.12 Public Beta

## Release Date: Jul. 20th 2026 00:30 JST

### Changelog
- Fixed an issue where the Emergency Connection Contract daily reward could not be claimed if players checked in before activating or renewing the contract.
- Fixed a display issue where the action bar overlapped the recipe list on the Manufacturing Station recipe page.
- Fixed an issue where the task icon on the Executor facility card could display incorrectly under certain conditions.
- Fixed an issue where the Data Affinity agent skill did not activate as intended, and added a new Package Specialist agent skill.
- Adjusted the maximum stack size of Neural Network Firewall Nodes to match Network Node Keys.
- Improved the recipe display in the Manufacturing Station.
- Limited-time dungeons no longer spawn Cache tiles in Boss rooms. Cache can now only be obtained by defeating the Boss.
- Limited-time dungeon sweeps now use a charge system (+1 charge/day, maximum 7, starts with 1/7) with adjusted Cache estimates to match.
- Moved Terminal User Network chats to the ChatNet page.
- Redesigned the loading animation.
- Redesigned the Regen Extraction page.
- Redesigned the slider in the General Number Input component.
- Improved the design of the Wasteland Development Terminal.

### Others
- Included various minor fixes and performance optimizations.
- Some balance adjustments applied to workstation_manufacturing_recipes.

***This is a hotfix update and will be installed automatically. You will receive a notification prompting you to refresh the page after the hotfix is installed.***
