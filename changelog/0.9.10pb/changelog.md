# Terminal Ark Changelog

## APP BUNDLE VERSION v0.9.10 Public Beta
## SERVER BUNDLE VERSION v0.9.10 Public Beta

## Release Date: Jul. 19th 2026 00:50 JST

### Changelog
- Fixed an issue where the bottom portion of the gacha buttons was hidden in the Signal Hijack System.
- Fixed an issue where Select Items was incorrectly labeled as Select Heroes in the Profile Editor.
- Fixed an issue where players could not edit heroes' equipment in the Profile Editor.
- Fixed an issue where players could not edit heroes on the /disc/team page.
- Improved the column view layout in the Inventory page.
- Added an empty-state hint for reward boxes with no items.
- Added a Dungeon Not Found message to the Join Dungeon page.
- Renamed the Black Market to Terminal Procurement.
- Introduced the Monthly Pass. Players can purchase the Emergency Link Contract for 128 Kib Cache to receive 1 Neural Link Stabilizer per day for 30 days.
- Redesigned Terminal Procurement.
- Adjusted the Neural Sync Rate cap to 180 (minimum) and 240 (maximum).
- Added a special dungeon tile that grants Cache.
- Removed Cache from general loot tiles.
- Fixed an issue where dungeon battle logs could display empty toast messages.
- Redesigned the Wasteland Development Terminal.

### Others
- Included various minor fixes and performance optimizations.
- Some balance adjustments applied to workstation_manufacturing_recipes.
