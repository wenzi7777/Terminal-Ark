# Terminal Ark Changelog

## APP BUNDLE VERSION v0.6.2 Public Beta
## SERVER BUNDLE VERSION v0.6.2 Public Beta

## Release Date: Apr. 3rd 2026 03:50 JST

### Changelog
- Added new "System Status" app in TerminalAppsMenu.
- Fixed an issue where TerminalBrowser has 2 scrollbars.
- Fixed an issue where TerminalBrowser might not update address bar in time.
- Fixed an issue where TerminalBrowser cant refresh page propolly.
- Added "Auto Arrange Team" and "Auto Equip Equipments" feature to help players arrange their teams.
- Equipment printing in the Molecular Assembler is now instant, consuming 1 matching blueprint and the required memory per item, then delivering gear and printing EXP directly to system mail.
- Major cache drops have been increased to roughly 2.2x to 2.5x of the previous rate, making dungeon clears and reward boxes a steadier source of cache.
- Resonance Hero Upgrade Core and Void Hero Upgrade Core were added, the base core recipe cost was reduced, EXP values were reworked, and hero leveling now supports mixed core usage.
- The Agent Workstation now includes the Execution Machine, which absorbs the old Crafting Station functions into categorized execution plans with byproducts, agent skill bonuses, power acceleration, per-second mood settlement, and an extra -5% execution time plus holo theme when connected to a Terminal User Network.
- The Agent Workstation now starts at Lv1 by default and auto-creates without needing the first compute unit, and the newbie tutorial now explicitly guides players into the workstation and execution machine.
- Skill upgrades for 3-star and 4-star heroes now cost only 1 matching rarity hero memory fragment per level.
- Same-attribute mitigation has been reduced from 50% to 20%, preventing same-type matchups from suppressing damage too heavily.
- All affected entry points, item use routes, locked package destinations, tutorial text, and item info were synchronized, and item text is now formally split into lore and description.
- Daily check-in has been replaced by monthly check-in, with a new JST-based monthly calendar generated each month and same-day-only reward claims.
- New personal amplifier plans were added to the Execution Machine, including the Blank Amplifier, Tactical Resonance Amplifier, and Aegis Phase Amplifier, with personal damage and defense buffs stackable up to 40%.
- New players now receive 20-minute Tactical Resonance and Defense Activation buffs on first login, granting +20% all-attribute damage and +20% all-attribute defense.
- Reworked the Regen Extraction page.
- Reworked the Black Market page.
- Reworked the Molecular Assembler page.
- Moved the receipes in Signal Tuner Hub to Craft Station in Agent Workstation.
- Reworked the Data Vault page.

### Others
- Included various minor fixes and performance optimizations.

***This is a hotfix update and will be installed automatically. You will receive a notification prompting you to refresh the page after the hotfix is installed.***
