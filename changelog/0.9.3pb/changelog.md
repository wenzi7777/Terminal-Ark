# Terminal Ark Changelog

## APP BUNDLE VERSION v0.9.3 Public Beta
## SERVER BUNDLE VERSION v0.9.3 Public Beta

## Release Date: Jul. 15th 2026 07:30 JST

### Changelog
- Fixed an issue where selected items and action buttons could overlap in the Resource Recycling execution plan of the Executing Machine in the Wasteland Development Terminal.
- Improved the agent dispatch workflow for facilities in the Wasteland Development Terminal.
- Improved the stability of the Wasteland Development Terminal.
- Removed Heal and Trap tiles from dungeons.
- Added a new Trap tile.
- Added a new Contaminated tile.
- Changed the dungeon room generation rules.
- Improved the UI of the Dungeon Viewer in the mobile view.
- Redesigned the Dialog Modal.
- Moved the Clear Working Area button to a more intuitive location.
- Redesigned the Home Action Button.
- Redesigned the Home Big Button and Fancy Button.
- Redesigned the Home Info Box, Small Overclock Box, and Small Chat Box.
- Redesigned the General Select component.
- Redesigned the Perform AFK page.
- Redesigned the personal and global buff UI.
- Fixed an issue where EXP calculations on the Perform AFK page could increase unexpectedly.
- Fixed an issue where particle effects remained enabled while Reduce Motion mode was turned on.
- Fixed an issue where some wrappers could scroll vertically unexpectedly.
- Redesigned the Home User Bar.
- Renamed the Team tab in /disc to Battle Preparation. Players can now create up to three battle profiles containing heroes and items, and activate one profile before entering a dungeon.
- Items can now be used as cards during battles.
- Removed item usage in dungeon rooms.

### Others
- Included various minor fixes and performance optimizations.
