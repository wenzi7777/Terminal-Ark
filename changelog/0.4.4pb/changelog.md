# Terminal Ark Changelog

## APP BUNDLE VERSION v0.4.4 Public Beta
## SERVER BUNDLE VERSION v0.4.4 Public Beta

## Release Date: Jan. 29th 2026 20:00 JST

### Bug Fixes
- Fixed an issue where “Use Support Hero” was not automatically disabled after the Terminal User Network connection was removed.
- Unfinished city battles will now resume after refreshing the page.
- Fixed various display issues with cards in the BattleView component.

### New Features
- Added the ability to quickly toggle between card details and quick play during battle.

### Improvements
- Removed the “Battle Preferences” button from the Battle View and merged the Deck Cards” and “Cooldown Cards” buttons into a single “Cards” button.
- You can now click the “Cards” button to choose between viewing Deck Cards” or “Cooldown Cards.”
- Improved the UI of "Card Details".
- The “Draw Card” and “Reduce AP” cards now include detailed descriptions and translations.
- Improved the values displayed on Cards, now values on Cards will be replaced with real calculated value, not reference value. You can still check reference value by viewing target card's details.
- Reworked the skill UI in the HeroDetails component.

### Others
- Included various minor fixes and performance optimizations.

***This is a hotfix update and will be installed automatically. You will receive a notification prompting you to refresh the page after the hotfix is installed.***
