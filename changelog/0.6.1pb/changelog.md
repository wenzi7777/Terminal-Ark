# Terminal Ark Changelog

## APP BUNDLE VERSION v0.6.1 Public Beta
## SERVER BUNDLE VERSION v0.6.1 Public Beta

## Release Date: Apr. 1st 2026 17:10 JST

### Changelog
- Reworked the "Next Turn Indicator" in Battle View.
- Fixed an issue where some cards could not be played during battle.
- Fixed an issue where clues could not be claimed after their cooldown had expired.
- Reworked the chat message input on the /chatnet page.
- Fixed an issue where the bottom spacer did not appear in the Battle View component in PWA apps.
- Reworked the hero leveling UI.
- Chat moderators and admins can now only apply personal buffs to players.
- Global buffs and personal buffs are now displayed on the profile page.
- Added an overall level system to indicate the highest dungeon level a player can enter.
- Updated AFK EXP calculation to be based on overall level.
- Applied balance adjustments to the new battle system.
- Updated onboarding rewards.
- Added overall level display to the AFK button in City mode.
- Introduced a new basic attack system in the battle system.

### Others
- Included various minor fixes and performance optimizations.
