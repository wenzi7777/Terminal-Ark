# Terminal Ark Changelog

## APP BUNDLE VERSION v0.5.17 Public Beta
## SERVER BUNDLE VERSION v0.5.17 Public Beta

## Release Date: Mar. 27th 2026 4:30 JST

### Bug Fixes
- Fixed an issue where the “Start Manufacture” and “Cancel” buttons in the Crafting Station of Agent Workstation overflowed off-screen.
- Fixed an issue where the overall layout width of Agent Workstation did not match the game’s design width.
- Fixed an issue where the facility upgrade overlay in Agent Workstation displayed incorrect information.
- Fixed an issue where buttons used to accelerate production with power lacked a loading animation.
- Fixed an issue where the Contact Center in Agent Workstation could not accelerate production when available power was below the full acceleration requirement.

### New Features
- Introduced Limited Time Dungeons, allowing players to obtain useful materials by clearing them.
- Introduced the Terminal Appearances feature, enabling players to apply skins visible to all players.
- Added Author and Chat Mod Terminal Appearances.

### Improvements
- Improved dungeon generation rules: each layer now guarantees at least one node that does not consume loot points.
- Agents can now be assigned to multi-slot facilities in a single action.
- Improved user group segmentation.
- Removed hover effects from hero skills.
- Removed hover effects from inventory filters.
- Introduced a redesigned user profile page with a more modern UI.
- Moved Online Players from the profile page to the Launchpad as a Terminal App.
- Removed the default avatar for players without a custom avatar.

### Others
- Included various minor fixes and performance optimizations.

***This is a hotfix update and will be installed automatically. You will receive a notification prompting you to refresh the page after the hotfix is installed.***
