# Terminal Ark Changelog

## APP BUNDLE VERSION v0.6.0 Public Beta
## SERVER BUNDLE VERSION v0.6.0 Public Beta

## Release Date: Mar. 31st 2026 3:20 JST

### Note
- Grouped changelogs (Bug Fixes, New Features, Improvements) will not be provided from this update.

### Changelog
- Clues can now be collected once the cooldown expires, regardless of agent assignment or mood (including when mood is 0).
- Fixed an issue where the deep_contract agent skill did not affect clue cooldown time.
- Added a new skills section to hero cards in the Battle View.
- Protagonist names are now centrally managed using template variables, with separate short and full display formats.
- Rebalanced combat damage, defense, enemy values, equipment feel, Memory economy, and onboarding resources under the new balance plan.
- Migrated hero classes to the new system with Vanguard; growth now fully follows a class × rarity table.
- Reworked hero skills.
- Improved enemy AI.
- Reworked the battle UI.
- Reworked the battle system.
- Updated the gacha system drop rate display page.
- Updated configurations for encrypted and standard channel pools.
- Improved the UI of the Signal Hijack system.
- Added a bottom spacer in PWA apps to prevent accidental taps on the system bar on mobile devices.
- Reworked the beginner tutorial to align with the updated battle system.
- Improved the display of enemy "next turn" skills on enemy cards.
- Improved the skill section display in enemy detail cards.
- Improved the buff/debuff display on hero cards.

### Others
- Included various minor fixes and performance optimizations.
