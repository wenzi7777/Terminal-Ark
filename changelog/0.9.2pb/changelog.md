# Terminal Ark Changelog

## APP BUNDLE VERSION v0.9.2 Public Beta
## SERVER BUNDLE VERSION v0.9.2 Public Beta

## Release Date: Jul. 13rd 2026 18:00 JST

### Changelog
- Removed the progress bar display for Proofs of Clue in the Wasteland Development Terminal.
- Added a Last Login display to the user profile page.
- Fixed an issue where equipment on the Resource Recycling execution plan selection page was not displayed correctly.
- Fixed an issue where EXP rewards from dungeon reward tiles did not scale with dungeon level.
- Added scrolling support for the equipment section in the reward preview overlay.
- Changed the color of General Resin from the Info color to the Rare color.
- Fixed an issue where the z-index of some facilities in the Wasteland Development Terminal was not set correctly.
- Auto-routing now treats all event tiles as blocked tiles.
- Fixed an issue where AP would automatically recover to 3 after being depleted while exploring dungeons.
- Reworked the UI of the Manufacturing Station in the Wasteland Development Terminal.
- Merged the Dispatch Agents feature into the Working Agents section. Players can now assign agents directly from the Working Agents section without switching to the Dispatch Agents tab.
- Reworked the UI of the Molecular Assembler.
- Fixed an issue where items could not be used in dungeons.
- Increased the item usage limit in dungeon boss rooms from 1 to 2. (The limit remains 1 in normal rooms.)
- Removed the Neural Sync Rate requirement of 40 for obtaining Cache.
- Redesigned the overlay style.
- Fixed an issue where the bottom spacer was missing on some pages in PWA mode.
- Fixed an issue where the height of tabs on the Preferences page was inconsistent in the mobile view.
- Redesigned the General Button component.
- Redesigned the channel selector in ChatNet.
- Redesigned the General Progress Bar component.
- Redesigned the hourglass loading animation.
- Redesigned the Secondary Progress Bar component.

### Others
- Included various minor fixes and performance optimizations.

***This is a hotfix update and will be installed automatically. You will receive a notification prompting you to refresh the page after the hotfix is installed.***
