# Terminal Ark Changelog

## APP BUNDLE VERSION v0.7.3 Public Beta
## SERVER BUNDLE VERSION v0.7.3 Public Beta

## Release Date: Jul. 6th 2026 2:30 JST

### Changelog
- Fixed an issue where item quantities in the Signal Retention Bay were displayed incorrectly.
- Fixed an issue where the item preview would not appear automatically when newly obtained items were added directly to the inventory.
- Added a notification explaining that duplicate heroes are converted into Memory Fragments.
- Refactored the Inventory page.
- Improved the text displayed in the Secondary Progress Bar for better readability.
- Replaced hardcoded style properties with CSS variables.
- Improved the BattleView component for both desktop and mobile players.
- Removed sensitive URLs and words from the error dialog.
- Fixed an overlap issue between the Terminal Appearance and the Channel Selector.
- Moved all GSAP animations to CSS animations except the Gacha animation.
- Reworked all Terminal Appearances, removed the scan line effect from all Terminal Appearances, and redesigned the Pioneer Terminal Appearance.
- Added the ChatNet overlay. Players can now click the Small Chat Box to open the ChatNet overlay and chat with other players while in battle.
- Redesigned the agent selection workflow in the Wasteland Development Terminal.

### Others
- Included various minor fixes and performance optimizations.

***This is a hotfix update and will be installed automatically. You will receive a notification prompting you to refresh the page after the hotfix is installed.***
