# Terminal Ark Changelog

## APP BUNDLE VERSION v0.6.9 Public Beta
## SERVER BUNDLE VERSION v0.6.9 Public Beta

## Release Date: Apr. 21st 2026 22:30 JST

### Changelog
- Fixed an issue that allowed multiple daily check-ins on the same day.
- Clicking outside the modal now triggers the "Done" action (no need to use the top-left button).
- Improved the game introduction page.
- Fixed a desktop UI issue where the Dialog component lacked border radius.
- Users are now redirected to the last visited page prior to logout upon login.
- Fixed dungeon death recovery flow: players are sent to the hospital first, then redirected back to the previous page after the countdown, with status preserved across reloads.
- Fixed an issue where neural sync icon in HomeUserBar will flash after neural sync is exceeding the limit.
- Adjust the quick extract buttons in the Link Recovery Guide and Residual Sync Extractor to use content-based sizing with proper padding instead of stretching to full width.

### Others
- Included various minor fixes and performance optimizations.
