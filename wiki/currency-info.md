# Currency Info (貨幣說明)

## 貨幣種類
- **Cache**：主要消耗貨幣，用於市集交易、製造、網路維護等。
- **Memory**：次要貨幣，用於儲存、升級、部分訂單與兌換。

## 取得方式
- **Cache**：市集販售、任務獎勵、地城/城市戰鬥掉落、網路維護收益。
- **Memory**：工作站訂單、地城/城市探索、活動獎勵、部分兌換。

## 消耗場景
- **Cache**：購買/升級裝備與設施、網路維護、代理人工作站相關支出。
- **Memory**：儲存於銀行、兌換特殊物資、部分升級或製造流程。

## 小提示
- 留意網路維護與製造成本，避免 Cache 見底。
- 適度將多餘 Memory 儲存或兌換，防止浪費。
- 遇到消耗提示時，先確認當前餘額與預期回報。
