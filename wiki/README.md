## 🎮 Game Features Overview | 遊戲功能概覽

### 🌐 Player Network System
A cooperative multiplayer feature where players can:
- Form networks with up to 10-30 members (based on network level)
- Share support heroes with network members
- Pool resources for network upgrades
- Chat and coordinate strategies

玩家網路系統是一個合作多人功能，玩家可以：
- 組建最多 10-30 人的網路（根據網路等級）
- 與網路成員分享助戰英雄
- 集合資源升級網路
- 聊天和協調策略

### ⚔️ Combat & Heroes
- **Turn-based card combat** with strategic depth
- **5 hero classes**: Striker, Tank, Healer, Mage, Support
- **Equipment system** with randomized affixes
- **Skill system** with upgradable abilities

### 💎 Resources
- **Memory (KiB)**: Common currency for upgrades and crafting
- **Cache**: Premium currency for advanced features
- **EXPD**: Experience points for hero leveling
- **Node Points**: Network resource for using support heroes
