# Tokens Info (代幣資訊)

本頁整理資源類代幣與「神經同步率」規則，方便快速查閱。

---

## 核心代幣 | Core Tokens

- **Memory (KiB)**
  - 常規資源，用於儲存、製造、升級與部分兌換。
- **Cache (KiB)**
  - 高階資源，用於市集交易、製造、網路維護與特殊流程。

---

## 取得方式 | Acquisition

- **Memory**：工作站訂單、地下城/城市探索、活動獎勵、部分兌換。
- **Cache**：
  - **AFK** 掉落（受神經同步率限制）
  - **地下城 Boss 房**：擊殺小額 + **Cache 訊號節點（地塊）** 為主要來源
  - 市集販售、任務獎勵、網路維護收益等

> 實際來源依版本調整，以遊戲內提示為準。

---

## 消耗場景 | Usage

- **Memory**：儲存、部分升級、兌換與製造流程。
- **Cache**：購買/升級裝備與設施、網路維護、代理人工作站相關支出。

---

## 神經同步率 | Neural Sync

神經同步率用於限制 **高價值獎勵是否完整收下** 的品質閘門（不足時降級為 fallback 或進入訊號保留艙），**不是**入場體力。

### 會消耗同步率的來源

| 來源 | 說明 |
|------|------|
| AFK 結算 | 稀有材料 + Cache |
| 地下城／城市 **戰鬥勝利** | 稀有材料、史詩／傳說裝備、以及獎勵中的 Cache |
| 地下城 **Cache 訊號節點** | 領取地塊上的 Cache |
| 探索 reward／污染格等 | 若掉落稀有材料則扣；**不再附帶 Cache** |
| 製造副產物、限時地下城特色掉落 | 依規則表 |
| 遠程執行地下城 | 與獎勵箱結算相同 |

### 地下城 Cache 發放（重點）

一般 enemy group **不再掉落 Cache**，避免一房多場戰鬥把同步率掏空。

| 項目 | 探索 | 挑戰 |
|------|------|------|
| 普通／精英 group 戰鬥 | Cache **0** | Cache **0** |
| Boss 擊殺小額 | 約 **1–2**（依等級插值） | 約 **2–4** |
| Boss 房 Cache 地塊數量 | **0–1** | **1–2**（保底 1） |
| 單格 Cache 量 | **1–5**（依等級插值） | **3–9** |
| 污染格／一般 reward 節點 | Cache **0** | Cache **0** |

地塊僅在 **Boss 房** 生成；需先清空敵人後才能領取。amount 在建本時寫入 metadata，領取時不再重 roll。

### 消耗規則（共通）

- **signal_tracer**：每個消耗 **30** 同步率；不足時可進訊號保留艙，過期改為 **brand_new_parts × 2**。
- **common_decoder**：每個消耗 **15** 同步率；不足時類似處理。
- **史詩／傳說裝備**：約 **60 / 50** 同步率；不足進保留艙。
- **Cache**：每 **1 Cache** 消耗 **10** 同步率；不足時改為 **brand_new_parts × 1**（Cache 本身不進保留艙）。

### 上限與恢復

- **基礎上限**：180
- **每 5 級 +10**
- **最大上限**：240（約 30 級達到）
- **恢復**：每分鐘結算，總量等效於每小時 +10
- **殘留同步率**：主條滿後繼續累積（上限為主上限一半），可萃取為穩定劑

---

## 小提示 | Tips

- 清普通房怪時，若沒掉稀有物，同步率應幾乎不動；高消耗集中在 **Boss 房 Cache 節點** 與稀有掉落。
- 滿條範圍為 **180–240**（隨等級成長，約 30 級封頂）；挑戰 Boss 房最壞約 220 同步率稅，低等級玩家請留意。
- 同步率不足時，稀有材料／裝備會先進入「訊號保留艙」，期限內可用同步率贖回。
- Cache 會被同步率直接限制，建議在領取 Boss 房 Cache 節點前確認同步率狀態。
- 同步率狀態也會影響首頁的「連線安全」提示。
