*Terminal Ark は、テキストを中核とした SF RPG で、気軽に長期間遊べる体験を重視しています。*

プレイヤーは自身のキャラクター、チーム、装備を管理し、物語の章を進めながらダンジョンを探索します。ソロプレイに加え、最大 2 人の他プレイヤーと協力して、カード型のターン制バトルを行うことも可能です。戦闘結果はキャラクタービルド、装備のアフィックス、戦術的な選択と密接に結びついており、あらゆる判断が戦局に影響を与えます。

本作は長時間のオフライン進行（AFK）に対応しています。長時間ログインしていなくても、資源・素材・経験値を継続的に獲得できます。次回ログイン時には、クラフト、取引、装備強化、アップグレードモジュールを通じて成長をさらに発展させ、自分だけの育成ルートを段階的に構築できます。装備には複数のレアリティとランダムなアフィックスの組み合わせが存在し、キャラクターの役割や戦闘スタイルに大きな影響を与えます。

また、ゲーム内には「ターミナル・ユーザー・ネットワーク」と呼ばれるギルドに近いシステムが存在し、志を同じくする仲間と一緒に遊ぶことができます。ダンジョン探索中には、「サポートヒーロー」システムを利用して、他のプレイヤーの強力なヒーローを借りることもでき、壮大な冒険を力強く支援してくれます。

世界には多数のランダム生成された敵、ダンジョン、探索エリアが存在します。その中でも多くのコンテンツは、プレイヤーコミュニティによって提供されたテンプレートやデータを基に構成されており、世界は常に拡張され、新鮮さを保ち続けます。さらに、多言語対応のチャットチャンネルが用意されており、すべてのプレイヤーが友好的でオープンな雰囲気の中で交流・共有・議論を行えます。

徹底した最適化を楽しみたい人も、世界観への没入を求める人も、あるいは時折ログインして進行状況を確認したい人も、Terminal Ark は自分のペースでリラックスしながら、着実に成長できる空間を提供します。

### バグ報告・ご提案

バグや技術的な問題が発生した場合は、このリポジトリで issue を作成してください。

機能追加の要望や一般的なご意見は、[Discord サーバー](https://discord.gg/jBAPuRuC) にて歓迎しています。

Terminal Ark の保守を手伝いたい方、新しいコンテンツを追加したい方、既存システムを改善したい方は、issue の作成や pull request の送信をお気軽にどうぞ。Discord で直接連絡することも可能です — wenzi7777。

### Terminal Ark をより良くするお手伝いをしませんか？

あらゆる形のコントリビューションを歓迎します：

- dungeon-lore/ —— ダンジョン探索中に表示される新しいロア（物語断片）を追加。
- heroes-collection/ —— ヒーローに関する Wiki 記事の執筆・改善。
- i18n/ —— UI 言語サポートの追加・改善。
- readme-i18n/ —— 本 README ファイルへの新しい言語対応の追加。
- stories/ —— メインストーリーの新しい言語対応を追加。
- ta-hints/ —— AFK 中に表示される新しいゲーム内ヒントを作成。
- wiki/ —— 一般的なゲームドキュメントの追加・拡充。
- equipment-names/ —— 新しい装備名を追加できます。プレイヤーが新しい装備を獲得した際に、ランダムで選択されます。
